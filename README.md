# Memopad

VS Code extension to save notes and prompts in the sidebar for quick access.

## Features

- Sidebar panel for quick access to notes
- Add, rename, delete notes
- Copy note contents to clipboard
- Auto-refresh when files change
- Notes stored in `~/.memopad`

## Usage

1. Click the Memopad icon in the activity bar
2. Click `+` to add a new note
3. Click on a note to open it
4. Right-click for copy, rename, or delete options

## Installation

Install from the VS Code marketplace or Cursor marketplace.
